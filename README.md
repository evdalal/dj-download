# dj-download-interface

This project helps DJs manage their online Spotify libraries with their downloaded files. It will sync their playlists from Spotify and download them locally.

### Instructions:

Use diy.py. Change username to spotify username. Create a .env file containing client id, redirect URL, authorization endpoint, response type, version, and client secret from Spotify Developer's dashboard. Run npm install and npm install-python-deps.
